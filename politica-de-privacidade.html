Política de Privacidade

Esta Política de Privacidade descreve como a IAssistent Tecnologia e Servicos de Comunicacao LTDA (“IAssistent”, “nós”, “nosso”) coleta, utiliza, armazena e protege os dados pessoais dos usuários de seus serviços, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD).

1. Quais dados coletamos

Podemos coletar dados pessoais como nome, e-mail, telefone, dados de navegação e informações fornecidas pelo usuário dentro da plataforma.

2. Finalidade da coleta

Utilizamos os dados para:

  • Fornecer e operar nossos serviços;
  • Melhorar a experiência do usuário;
  • Cumprir obrigações legais;
  • Comunicar informações importantes sobre o uso da plataforma.

3. Compartilhamento de dados

Não compartilhamos dados pessoais com terceiros, exceto:

  • para cumprir determinações legais;
  • quando necessário para prestação do serviço;
  • quando autorizado pelo titular.

4. Segurança

Adotamos medidas técnicas e administrativas adequadas para proteger os dados pessoais contra acessos não autorizados.

5. Direitos do usuário

Em conformidade com a LGPD, o usuário pode solicitar:

  • Confirmação do tratamento;
  • Acesso aos dados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Eliminação dos dados pessoais tratados com consentimento;
  • Portabilidade dos dados;
  • Revogação do consentimento.

6. Contato para privacidade

Para exercer seus direitos, entre em contato pelo e-mail: privacidade@iassistent-tech.com.br.
